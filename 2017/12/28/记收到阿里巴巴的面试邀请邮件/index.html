记收到阿里巴巴的面试邀请邮件

在前天中午，突然收到一封邮件，本以为有何平常一样是个广告，然而我却看到了我在github上的名字，觉得有可能是有人对我的项目感兴趣给我发邮件了，没想到仔细一看居然是阿里旗下的菜鸟网络发过来的。

这真是让我太惊喜了，为了防止假消息我还特意访问了发件人邮件的地址，这真的是来自于阿里的菜鸟部门。这让我受宠若惊，因为菜鸟发现了我在 Github 上的项目然后给我发的邮件，因为上面的项目我已经没有去维护了，都是半年前为了熟悉新的技术写的小软件，很多地方都存在不足。

在收到邮件后，还是万分激动的，因为这意味着有人认可我，特别还是来自 BAT 的其中之一，让我去参加面试。这让我受到了莫大的鼓舞，让我更加的有信心。再此我以后一定会好好维护我的 Github 项目，分享更多有用的知识。我的实力也会越来越强的，加油！
