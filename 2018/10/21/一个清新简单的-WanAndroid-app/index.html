一个清新简单的 WanAndroid app



Introduce

一个基于 WanAndroid Api 完成的第三方 app。坚持做一个简单好用纯粹的 app。 Simple is more

代码结构为：单 Actvity + Fragment && Retrofit + RxJava + MVP +Dagger2

Github 上的 WanAndroid 总是多多少少让我觉得不够满意，索性自己设计一个简单并且好用的，顺便熟悉一下最新的开源库。希望你也能喜欢！

如果有任何问题，请到github的issue处提交，或者联系我，感谢！
